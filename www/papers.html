CSlib Papers

If you use the CSlib library in one of your publications, please cite the following paper and also list the URL of the CSlib website, namely http://cslib.sandia.gov.

This paper describes the MR-MPI library and several MapReduce graph algorithms for informatics problems:

MapReduce in MPI for Large-Scale Graph Algorithms, S. J. Plimpton and K. D. Devine, Parallel Computing, 37, 610-632 (2011). (abstract) (preprint)

The author of the MR-MPI library are Steve Plimpton (sjplimp at sandia.gov) and Karen Devine (kddevin at sandia.gov).

You can send us email regarding any of these issues:

  • If you use MR-MPI for something interesting, we'd like to hear about it.
  • If you write a paper using the MR-MPI library, we can post a citation to it on this page.
  • If you write a MapReduce program which calls the library, that others would be interested in, you can send it and we'll include it in the distribution.
  • If you enhance the library itself or have ideas for doing so, we can discuss including the improvements in future releases.
