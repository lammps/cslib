CSlib Papers

If you use the CSlib library in one of your publications, please refernence the URL of the CSlib website, namely http://cslib.sandia.gov.

At some point we may write a paper about use of the CSlib, in which case we will post a citation here.

The author of the CSlib is Steve Plimpton (sjplimp at sandia.gov).

You can send me an email regarding any of these issues:

  • If you use the CSlib for something interesting, I'd like to hear about it.
  • If you write a paper using the CSlib, I can post a citation to it on this page.
  • If you enhance the CSlib or have ideas for doing so, we can discuss including the improvements in future releases.
